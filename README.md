# Books Catalog

Project to compare the TAG books ratings with the GoodReads api ratings

## Setup

> Download the source code from (Github Repository)[https://github.com/]

Access from your terminal the `backend folder` and install the dependencies

```bash
$ npm i
```

> For running your project, locally, go inside the root of the frontend folder and rename the file `.env.example` to `.env.local`. This file contains the URL path for the backend api calls

Install the front-end dependencies, by openning your terminal in the `frontend folder` and executing the following command

```bash
$ npm i
```

---

## Usage

Access from your terminal the `backend folder` and run the following

```bash
$ npm start
```

Open a new terminal without closing the other terminal, currently, running the backend. Now inside the `frontend folder` execute

```bash
$ npm start
```
